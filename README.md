## General Installation

- Follow the installation process [here](https://lorisleiva.com/create-a-solana-dapp-from-scratch/getting-started-with-solana-and-anchor).
    - Install Rust
    - Install Solana
    - Install NVM
    - Install Node.js
    - Install Mocha
    - Install Yarn
    - Install Anchor
- For each project folder, install using 
    ```
    npm i
    ```
